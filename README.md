# IntelliJ Hackatime
The Wakatime IntelliJ plugin is kinda buggy sometimes, so here is a solution
to a very specific problem I have with a very specific problem! :D

With some luck I won't be the only person that finds this useful...

(It's basically a recode on Kotlin of the already existing WakaTime plugin, so yeaaaaah...)

# How to use
To install the plugin, follow these easy steps:

1. Download the [latest release](https://github.com/dorythecat/IntelliJHackatime/releases/tag/release) from the GitHub.
   <img width="1600" height="668" alt="Screenshot 2025-11-12 at 20-02-21 Release v1 0 Â· dorythecat_IntelliJHackatime" src="https://github.com/user-attachments/assets/d06e1aa2-4da0-4ffb-9936-1b577e5bcbd6"/>
3. Open up your preferred Jetbrains IDE.
4. Click `Ctrl+Alt+S` to open the settings menu (Or go to File -> Settings on the IDE)
6. Go to `Plugins`. It should look like this:
   <img width="1247" height="939" alt="Captura de pantalla 2025-11-12 200546" src="https://github.com/user-attachments/assets/e6236216-9594-4966-8a86-03d56087e877" />
8. Click on the settings wheel inside this menu, and then on `Install Plugin from from Disk...`.
   <img width="354" height="328" alt="Captura de pantalla 2025-11-12 200701" src="https://github.com/user-attachments/assets/cca4762f-1409-4d86-a69f-6fe9b706108f" />
10. Select the Plugin's jar.
11. All set! :D

The program is a bit buggy at the time, but hopefully it works right for ya, and if not, please open an Issue. Thank you in advance. <3
